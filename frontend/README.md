# Frontend Structure

This project currently serves two static UI pages from the repository root:

- `landing.html` - project landing page
- `app.html` - bridge application UI

The `frontend/` directory is the dedicated home for frontend-related files going forward.

## Local run

From the project root:

```bash
npm run frontend:dev
```

Then open:

- `http://localhost:5173/landing.html`
- `http://localhost:5173/app.html`
- `http://localhost:5173/frontend/` (frontend module entry)
